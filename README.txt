Welcome to the room request program!
1. To get started, make sure that Requester.c and RequestHandler.c are compiled. 
2. Be sure to include the proper tags to compile RequestHandler with pthreads
3. Run RequestHandler. When prompted, you will be allowed to open a thread to enter data by pressing any key.
4. The thread will appear in another terminal window. Enter input materials in there, and press any key to continue
	when prompted. Please enter input as prompted. 
5. Output will appear in the RequestHandler window in near-real time.
